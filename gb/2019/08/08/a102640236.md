<tr><td VALIGN=TOP width="626"><h2 align=center>三星Galaxy Note 10系列正式亮相 推5G版</h2>
<img src="https://i.ntdtv.com/assets/uploads/2019/08/1-118-800x450.jpg" />
<h6>Galaxy Note 10有着非常独特的表面处理，拿在手中，Galaxy Note 10手机会在不同光线下改变颜色。（TIMOTHY A. CLARY/AFP/Getty Images)</h6>
<hr>
	<p>【新唐人北京时间2019年08月08日讯】<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/三星.md">三星</a>在周三（8月7日）发布了三款新的高端智能手机：<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/galaxy-note-10.md">Galaxy Note 10</a>,Galaxy Note 10+和 Galaxy Note 10+ 5G。这三款起价为950美元的新机型更薄、更轻、更具非凡动力，并改变了Galaxy Note手机的阵容。</p>
<p>据美国全国广播公司财经频道（CNBC）报导，现在发布新手机是<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/三星.md">三星</a>赶在<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/苹果.md">苹果</a>公司将于今年秋季发布新款iPhone之前，在市场上推出自己最新、最好的手机的一次难得机会。在三款Note 10手机中，每个人都会找到自己喜欢的那一款，但前提是，你愿意花至少950美元去把它拿到手。</p>
<p>人们愿意花1,000美元去买一部手机的市场正在萎缩，因为许多人比以往任何时候都更加愿意继续使用自己已经拥有的手机。这意味着，在<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/苹果.md">苹果</a>于9月份发布新款iPhone之前，三星推出新款Note手机并抢先吸引这部分可能会动心的消费者是非常重要的。</p>
<p>此次也是三星首次同时推出了多款一直被粉丝们所喜爱的Galaxy Note手机，这显然会有助于满足尽可能多的消费者的不同需求。对于那些不喜欢“超大”手机的人来说，有一款较小的机型；对于高级用户来说，有一款较大的机型；对于那些希望不会因下一代网络而导致手机过时的人士来说，还有一款5G的机型。三款新手机都将马上被推出。</p>
<p>三星宣布说，这三款手机将于美国东部时间周四（8月8日）凌晨12:01开始订购，并将于8月23日在店铺销售。5G版本手机将会是Verizon公司的独家版本。<div class="video_fit_container"><a width="764" b="430" class="video_frame" src=""></a><a href="https://git.io/fjQje"><img width="600" src="https://raw.githubusercontent.com/asdfgt5/1/master/t/ntdtv/ntdtv600.jpg" title="进入原文观看"  alt="进入原文观看"></a><h3 align=center>(视频.国内建议下载<a href="https://git.io/JesJV">翻墙软件</a>进入原文观看)</h3><hr><a src="https://www.youtube.com/embed/CYA_XAMfRfo?wmode=transparent&#038;wmode=opaque" allowfullscreen></a>
</div></p>
<p>下面是关于这三款Galaxy Note新手机的一些信息，希望能为你的选购提供帮助：</p>
<p><strong><a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/galaxy-note-10.md">Galaxy Note 10</a><br />
</strong><br />
Galaxy Note 10被认为是大多数人应该购买的手机。这款 Galaxy Note 10有着非常独特的表面处理，拿在手中，Galaxy Note 10手机会在不同光线下改变颜色。</p>
<p>这款售价为950美元的三星Galaxy Note 10手机的屏幕为6.3英寸，这是它与屏幕为6.8英寸的Galaxy Note 10+ 机型之间的最大区别。三星表示，他们还正在推出一款更小的Note手机，以保证那些认为全尺寸的Note手机太大的消费者们满意。</p>
<p>无疑，这是一个很好的营销策略，因为早期的Note手机的确被一些人认为是有点儿“笨重”。但今年的新款Galaxy Note 10手机给人的感觉则是更轻、更薄。手机尺寸仍然很大，但由于屏幕占据了手机前部面板的大部分空间，所以它的实际尺寸已经不像早期型号那么大了。</p>
<p>新的屏幕是一个很大的卖点。这可能是你能够买到的最好的手机屏幕了。它色彩鲜艳、对比鲜明。</p>
<p>手机表面的处理也非常漂亮。这些都可能会使之成为目前市场上最好看的手机。其中，银色的“光芒光环”（Aura Glow）一定会被很多人所喜欢。</p>
<p>后面有三个摄像头，它们给人的感觉很不错。一个会给你两倍光学变焦，另一个是广角镜头，你不用再单独买个广角镜头套在手机摄像头上了。还有另一种标准相机，当与光学变焦镜头配合使用时，可以让你拍摄肖像照片。</p>
<p>S Pen（请不要叫它铁笔）也得到了改善。它在人体工程学方面进行了一些改进，使之在屏幕上书写会更加自然。此外，它给人的感觉也更加坚固了，有斜面的边会使抓握更容易。尽管一些人仍会认为它还是有点小。</p>
<p>新推出的一些S Pen新功能包括了手势支持，所以你可以不用触摸手机就可以对之进行设置，可以控制相机的变焦等等。虽然这听上去有点噱头的嫌疑，但实际上，三星的确一直都在努力吸引那些可能喜欢这些看起来高大上的小功能的资深用户。</p>
<p>三星手机一直致力于使用更大的电池和更快的充电。今年，新手机取消了标准的耳机插孔，以便尽可能地挤进更大块的电池。该公司表示，消费者可以在一小时内给 Galaxy Note 10充满电，或者在充电30分钟后使用一天。这听上去的确很不错。手机充电的速度越快，对电池寿命的担心就会越少。</p>
<p>最后，您会获得相当于入门级iPhone手机四倍的存储空间。三星发布的 Note 10拥有256GB 的存储，这已经远远超过了大多数人的需求。但是它没有Micro SD卡插槽，所以不能像以前的Note机型那样另外增加存储空间。</p>
<p><strong>Galaxy Note 10+<br />
</strong><br />
售价1,099美元的Galaxy Note 10+是Galaxy Note 10手机的加大版本，但同时，它也有一些重要的改变，也许这些改变将会更吸引Note的铁杆粉丝。</p>
<p>首先，它有一个更大的6.8英寸的屏幕。虽然屏幕很大，但手机却又轻又薄，所以不会令人觉得太笨重。当然，它绝对不是一个可以单手使用全套功能的手机。</p>
<p>Note 10+有一个更大的电池，可以使用256GB 或512GB的存储空间，这本身都足够大了，但它还有一个Micro SD卡插槽，所以你可以为之添加更多的存储空间。这对那些安装了很多游戏，下载了很多电影，重要的是拍摄了很多高分辨率视频的人来说会很有吸引力。用S Pen编辑视频会是非常有趣的过程，你会更容易地点击和选择一个镜头中的具体某一帧。</p>
<p>Galaxy Note 10+将包括高达12GB的内存，而Galaxy Note 10的内存为8GB。这意味着在同时运行大量应用程序的情况下，Note 10 +的性能会更平稳，包括在三星的“Dex”桌面环境中。该环境允许你连接显示器、鼠标和键盘，并将手机作为一种迷你PC使用。</p>
<p>“Dex”还可以让你将手机插入PC 或Mac，并在安全的窗口环境下运行手机上的应用程序。以前，你必须把它连接到显示器上，这会令休闲和游戏更为方便。</p>
<p>Note 10+的背面还有一个“飞行时间”（Time of Flight）传感器，它可以使用光束来更好地测量与物体之间的距离。目前，这主要用于拍摄出更好的人像照片，但它也可以增强实境应用程序和游戏。现在它还不是一个会被经常使用的功能，但是将来随着更多的增强现实应用的出现，它可能就会变得非常有用。</p>
<p>除此之外，Galaxy Note 10+的三个摄像头等其它功能与标准版相同。</p>
<p><strong>Galaxy Note 10+ 5G</strong></p>
<p>Verizon将在早期独家销售 Galaxy Note 10+ <a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/5g手机.md">5G手机</a>，顾名思义，该手机支持5G网络。除此之外，它几乎与Galaxy Note 10+完全相同。如果你不认为你会在未来三年或四年内进一步升级你的手机，而且你还有不少的余钱要花掉的话，那么这款5G的Note 10手机就是为你准备的。</p>
<p>但是，也有观点认为，现在并不需要急于购买<a href="https://github.com/asdfgt5/ntdtv/blob/master/gb/5g手机.md">5G手机</a>。Verizon公司表示，即使到2020年，其5G网络将只会覆盖美国一半的地区，目前只覆盖少数城市的有限地区。Galaxy Note 10+ 5G将以1,299.99美元的价格出售256GB版本，以及1,399.99美元的512GB版本。对于一个你在大多数地方都还不能使用5G的手机来说，这真不能算是一个很小的数目。</p>
<p><strong>最后的一些想法<br />
</strong><br />
总之，Note 10和 Note 10+新手机提供了几乎所有你想从Android手机中获得的功能，包括屏幕内指纹识别器、无线快速充电、支持其它设备无线充电以及防水、防尘等等等等。</p>
<p>三星的One UI应用程序也仍然处于业界前沿和中心位置，当它第一次在Galaxy S10上发布时，相对于老版本的软件就已经是一个很大的改进。</p>
<p>无疑，此次三星发布三款新手机已经抢了苹果公司9月份发布会的先机。在外界对苹果的新手机外观和性能仍一无所知的情况下，可能很多人都会先入为主地选择标准已经很高的三星的Note 10手机。而那些本来就是三星Note手机粉丝的人士，在拿到它后，更是会对之惊喜连连。</p>
